# 포트폴리오 프로젝트 실행 방법

## 방법 1: 기본 서버 실행 (현재 설정)

### 1단계: 터미널 열기
- 프로젝트 폴더에서 우클릭 → "터미널에서 열기"
- 또는 VS Code에서 `Ctrl + `` (백틱)으로 터미널 열기

### 2단계: 서버 실행
```bash
npm run dev
```
또는
```bash
npm start
```

### 3단계: 브라우저에서 접속
```
http://localhost:3000
```

**주의사항**: 파일을 수정한 후에는 브라우저에서 수동으로 새로고침(F5)해야 합니다.

---

## 방법 2: 자동 새로고침 (라이브 리로드) - 추천!

### 2-1. nodemon 사용 (서버 자동 재시작)

#### 설치:
```bash
npm install --save-dev nodemon
```

#### package.json 수정:
`"dev": "node server.js"` → `"dev": "nodemon server.js"`로 변경

#### 실행:
```bash
npm run dev
```

**효과**: 파일 변경 시 서버가 자동으로 재시작됩니다. (브라우저는 여전히 수동 새로고침 필요)

---

### 2-2. 완전한 라이브 리로드 (브라우저 자동 새로고침)

#### 설치:
```bash
npm install --save-dev nodemon
```

#### 실행 스크립트 수정:
`package.json`의 `dev` 스크립트를 `nodemon server.js`로 변경

#### index.html에 자동 새로고침 스크립트 추가:
`</body>` 태그 앞에 다음 코드 추가:
```html
<script>
  // 개발 모드에서만 작동하는 자동 새로고침
  if (window.location.hostname === 'localhost' || window.location.hostname === '127.0.0.1') {
    const ws = new WebSocket('ws://localhost:3001');
    ws.onmessage = () => location.reload();
  }
</script>
```

#### server.js에 WebSocket 서버 추가 필요
(복잡하므로 간단한 방법은 아래 참고)

## 방법 3: 가장 간단한 라이브 리로드 (권장)

### Live Server 확장 프로그램 사용 (VS Code)

1. VS Code에서 확장 프로그램 설치:
   - `Ctrl + Shift + X` → "Live Server" 검색 → 설치

2. `index.html` 파일에서 우클릭 → "Open with Live Server"

**장점**: 
- 별도 설정 불필요
- 파일 저장 시 자동 새로고침
- 가장 간단함

---

## 방법 4: Python 간단 서버 (Node.js 없이)

```bash
# Python 3가 설치되어 있다면
python -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속

---

## 현재 프로젝트 상태

- ✅ 서버 파일 (`server.js`) 준비됨
- ✅ package.json 스크립트 설정됨
- ⚠️ 라이브 리로드 기능 없음 (수동 새로고침 필요)

## 추천 실행 순서

1. **지금 바로 실행**: `npm run dev` → `http://localhost:3000` 접속
2. **자동 새로고침 원하면**: VS Code Live Server 확장 사용
3. **더 많은 기능 원하면**: nodemon 설치 후 사용

## 문제 해결

### 포트가 이미 사용 중일 때
`server.js` 파일에서 `PORT` 값을 3001, 3002 등으로 변경

### npm 명령어가 작동하지 않을 때
```bash
npm install
```
먼저 실행 후 다시 시도
